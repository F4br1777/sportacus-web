# Sportacus: Gestión Deportiva Inteligente 🏋️‍♂️

Sistema Web integral diseñado para la administración eficiente de gimnasios y centros deportivos.

## 🚀 Funcionalidades Clave
* **Control de Cuotas:** Cálculo automático de intereses y estados de pago.
* **Métricas Físicas:** Seguimiento detallado de la evolución de los alumnos.
* **Notificaciones:** Envío automatizado de avisos por WhatsApp.
* **Arquitectura:** Solución serverless conectada a Google Sheets mediante Apps Script.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend/DB:** Google Apps Script & Google Sheets API.
* **Integraciones:** WhatsApp Web API.
